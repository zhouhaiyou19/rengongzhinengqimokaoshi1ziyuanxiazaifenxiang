      2. **连接主义（Connectionism）**：
         - 例：深度神经网络
            - 连接主义主张通过模拟人脑神经网络的结构和功能来实现智能。深度神经网络是连接主义的代表，它通过多层神经元网络进行复杂的模式识别和数据处理。

            3. **行为主义（Behaviourism）**：
               - 例：机器人
                  - 行为主义关注智能体与环境的交互行为，强调通过学习和适应来实现智能。机器人是行为主义的一个典型应用，它通过感知环境、做出决策并执行动作来完成任务。
